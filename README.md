# EEG Band Power Analysis

This repository contains a **Jupyter Notebook** demonstrating EEG band power analysis using Python and the **MNE** library.

## Notebook

- `notebooks/01_eeg_band_power.ipynb` — Loads sample EEG data from MNE, computes the **Power Spectral Density (PSD)**, and visualizes power in standard EEG frequency bands:
  - Delta (0.5–4 Hz)
